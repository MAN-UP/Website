Talks

As a society we run weekly talks about all things computer science. Past talks have inclued

  • How to program your house with arduinos
  • How to build mobile apps people want to use
  • Automatically generated game content
  • What actually happens inside the JVM (Java Virtual Machine)
  • How to visualize the 4th dimension
  • What's functional programming and what's it good for?
We're afraid that one project that we're currently working on is this website! In fact, if you'd like to get involved and help us improve it (and add that to your CV!) then please do contact us.

At one point in the past we recorded all of our talks - and these recordings can be found here. However, we stopped doing this at one point but this year we're going to start again. If this is something you'd be interested in being involved in let us know!
